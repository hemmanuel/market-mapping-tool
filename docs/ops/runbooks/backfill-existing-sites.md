# Backfill Existing Sites

## Rule

Backfill existing sites by seeding new runs, not by bypassing acquisition stages.

## Procedure

1. Choose site IDs and backfill objective.
2. Create new run records for those sites.
3. Seed the run with the correct front-door starting task.
4. Let the orchestrator execute discovery, source acquisition, storage, and projection.
5. Review yields, preserved artifacts, and graph updates.
